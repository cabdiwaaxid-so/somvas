Somvas Framework

A lightweight, modular framework for creating 2D games that work on both mobile and desktop devices.

Features

  • Canvas rendering with game loop management
  • GameObject and component system
  • Asset loading for images and audio
  • Input handling for keyboard, mouse, and touch
  • Collision detection with Box and Circle colliders
  • Scene management system
  • Math utilities for game development

Quick Start

Get started with Somvas in three simple steps:

1. Include the framework

<script src="https://cdn.jsdelivr.net/gh/cabdiwaaxid-so/somvas@main/index.js"></script>

2. Create HTML canvas

<canvas id="gameCanvas"></canvas>

3. Initialize your game

// Initialize the framework
Somvas.init('gameCanvas', 800, 600);

// Create a scene
const mainScene = Somvas.createScene('main');

// Create a game object
const player = new Somvas.GameObject(400, 300);
player.addComponent(new Somvas.Rectangle(50, 50, 'blue'));
mainScene.add(player);

// Set and start the scene
Somvas.setScene('main');
Somvas.start();

Installation

Browser (CDN)

Include the framework directly in your HTML file:

<!-- Latest version from CDN -->
<script src="https://cdn.jsdelivr.net/gh/cabdiwaaxid-so/somvas@main/index.js"></script>

<!-- Or download and include locally -->

Core Concepts

Initialization

Initialize the framework with a canvas element:

// Initialize with existing canvas element
Somvas.init('gameCanvas');

// Or create a new canvas with custom dimensions
Somvas.init(null, 800, 600);

GameObject

GameObjects are the basic entities in your game. They can have position, rotation, scale, and components:

// Create a new GameObject at position (100, 200)
const obj = new Somvas.GameObject(100, 200);

// Set properties
obj.rotation = Math.PI / 4; // 45 degrees
obj.scaleX = 2;
obj.scaleY = 0.5;
obj.visible = true;

Components

Components add functionality to GameObjects. Built-in components include Sprite, Text, Rectangle, and Circle:

// Add a rectangle component
obj.addComponent(new Somvas.Rectangle(50, 50, 'red'));

// Add a text component
obj.addComponent(new Somvas.Text('Hello World', {
    font: '20px Arial',
    fill: 'white'
}));

// Add a sprite component (after loading image)
const sprite = new Somvas.Sprite(Somvas.AssetLoader.get('playerImage'), 64, 64);
obj.addComponent(sprite);

Scenes

Scenes manage collections of GameObjects and can have backgrounds:

// Create a scene
const scene = Somvas.createScene('main');

// Set background color
scene.background = '#87CEEB'; // Sky blue

// Add game objects to the scene
scene.add(player);
scene.add(enemy);

// Set as current scene
Somvas.setScene('main');

Asset Loading

Load images and audio files with the AssetLoader:

// Load a single image
Somvas.AssetLoader.loadImage('player', 'assets/player.png')
    .then(() => {
        console.log('Player image loaded');
    })
    .catch(error => {
        console.error('Error loading image:', error);
    });

// Load multiple images
Somvas.AssetLoader.loadImages({
    player: 'assets/player.png',
    enemy: 'assets/enemy.png',
    background: 'assets/bg.jpg'
}).then(() => {
    console.log('All images loaded');
    // Start your game here
});

Input Handling

Handle keyboard, mouse, and touch input:

// In a component's update method
update(deltaTime) {
    // Keyboard input
    if (Somvas.Input.isKeyPressed('ArrowRight')) {
        this.gameObject.x += 5;
    }
    
    // Mouse input
    if (Somvas.Input.isMousePressed()) {
        const mousePos = Somvas.Input.getMousePosition();
        // Do something with mouse position
    }
    
    // Touch input
    if (Somvas.Input.isTouchPressed()) {
        const touchPos = Somvas.Input.getTouchPosition();
        // Do something with touch position
    }
}

Collision Detection

Add colliders to GameObjects for collision detection:

// Add a box collider
player.setCollider(Somvas.BoxCollider, 50, 50);

// Add a circle collider
bullet.setCollider(Somvas.CircleCollider, 10);

// Check for collisions in update
update(deltaTime) {
    for (const enemy of this.scene.find('enemy')) {
        if (this.gameObject.checkCollision(enemy)) {
            // Handle collision
            console.log('Collision detected!');
        }
    }
}

API Reference

Main API Methods

Somvas.init(canvasId, width, height)

Initializes the framework with a canvas element.

Parameters: canvasId (string|undefined), width (number|undefined), height (number|undefined)

Returns: void

// Initialize with existing canvas
Somvas.init('gameCanvas');

// Create new canvas with dimensions
Somvas.init(null, 800, 600);

Somvas.start()

Starts the game loop.

Parameters: none

Returns: void

// Start the game
Somvas.start();

Somvas.stop()

Stops the game loop.

Parameters: none

Returns: void

// Stop the game
Somvas.stop();

Somvas.createScene(name)

Creates a new scene with the given name.

Parameters: name (string)

Returns: Scene instance

// Create a scene
const mainScene = Somvas.createScene('main');

Somvas.setScene(name)

Sets the current active scene.

Parameters: name (string)

Returns: void

// Set the active scene
Somvas.setScene('main');

Somvas.toggleDebug()

Toggles debug mode which visualizes colliders.

Parameters: none

Returns: void

// Toggle debug mode
Somvas.toggleDebug();

GameObject

new GameObject(x, y, layer)

Creates a new GameObject at the specified position.

Parameters: x (number), y (number), layer (number, default: 0)

Returns: GameObject instance

// Create a new GameObject
const obj = new Somvas.GameObject(100, 200, 1);

addComponent(component)

Adds a component to the GameObject.

Parameters: component (Component instance)

Returns: this (for chaining)

// Add a rectangle component
obj.addComponent(new Somvas.Rectangle(50, 50, 'blue'));

getComponent(type)

Gets a component of the specified type from the GameObject.

Parameters: type (Component class)

Returns: Component instance or undefined

// Get the Sprite component
const sprite = obj.getComponent(Somvas.Sprite);

setCollider(type, ...args)

Adds a collider to the GameObject.

Parameters: type (Collider class), ...args (collider-specific parameters)

Returns: Collider instance

// Add a box collider
obj.setCollider(Somvas.BoxCollider, 50, 50);

// Add a circle collider
obj.setCollider(Somvas.CircleCollider, 25);

checkCollision(other)

Checks if this GameObject collides with another GameObject.

Parameters: other (GameObject)

Returns: boolean

// Check collision with another GameObject
if (player.checkCollision(enemy)) {
    console.log('Collision detected!');
}

Components

Rectangle(width, height, color)

Creates a rectangle component with specified dimensions and color.

Parameters: width (number), height (number), color (string)

Returns: Rectangle component instance

// Create a red rectangle
const rect = new Somvas.Rectangle(100, 50, 'red');

Circle(radius, color)

Creates a circle component with specified radius and color.

Parameters: radius (number), color (string)

Returns: Circle component instance

// Create a blue circle
const circle = new Somvas.Circle(30, 'blue');

Sprite(image, width, height)

Creates a sprite component with an image.

Parameters: image (Image|string), width (number), height (number)

Returns: Sprite component instance

// Create a sprite from loaded image
const sprite = new Somvas.Sprite(Somvas.AssetLoader.get('player'), 64, 64);

// Or directly from image URL
const sprite2 = new Somvas.Sprite('assets/player.png', 64, 64);

Text(content, style)

Creates a text component with specified content and style.

Parameters: content (string), style (object)

Returns: Text component instance

// Create a text component
const text = new Somvas.Text('Hello World', {
    font: '20px Arial',
    fill: 'white',
    align: 'center'
});

AssetLoader

AssetLoader.loadImage(key, url)

Loads a single image and stores it with the specified key.

Parameters: key (string), url (string)

Returns: Promise

// Load a single image
Somvas.AssetLoader.loadImage('player', 'assets/player.png')
    .then(() => console.log('Image loaded'))
    .catch(err => console.error('Error loading image:', err));

AssetLoader.loadImages(images)

Loads multiple images specified as key-url pairs.

Parameters: images (object)

Returns: Promise

// Load multiple images
Somvas.AssetLoader.loadImages({
    player: 'assets/player.png',
    enemy: 'assets/enemy.png',
    background: 'assets/bg.jpg'
}).then(() => console.log('All images loaded'));

AssetLoader.get(key)

Retrieves a loaded asset by its key.

Parameters: key (string)

Returns: Image or undefined

// Get a loaded image
const playerImage = Somvas.AssetLoader.get('player');

Input

Input.isKeyPressed(key)

Checks if a key is currently pressed.

Parameters: key (string)

Returns: boolean

// Check if arrow key is pressed
if (Somvas.Input.isKeyPressed('ArrowRight')) {
    player.x += 5;
}

Input.isMousePressed()

Checks if the mouse button is currently pressed.

Parameters: none

Returns: boolean

// Check if mouse is pressed
if (Somvas.Input.isMousePressed()) {
    // Do something
}

Input.getMousePosition()

Gets the current mouse position relative to the canvas.

Parameters: none

Returns: {x: number, y: number}

// Get mouse position
const mousePos = Somvas.Input.getMousePosition();
console.log(`Mouse at: ${mousePos.x}, ${mousePos.y}`);

Input.getTouchPosition()

Gets the current touch position relative to the canvas.

Parameters: none

Returns: {x: number, y: number}

// Get touch position
const touchPos = Somvas.Input.getTouchPosition();
console.log(`Touch at: ${touchPos.x}, ${touchPos.y}`);

Examples

Hello World Example

A minimal example that creates a canvas with a blue rectangle.

<!DOCTYPE html>
<html>
<head>
    <title>Somvas Hello World</title>
    <script src="https://cdn.jsdelivr.net/gh/cabdiwaaxid-so/somvas@main/index.js"></script>
</head>
<body>
    <canvas id="gameCanvas"></canvas>
    
    <script>
        // Initialize the framework
        Somvas.init('gameCanvas', 800, 600);
        
        // Create a scene
        const scene = Somvas.createScene('main');
        scene.background = '#f0f0f0';
        
        // Create a game object with a rectangle component
        const rect = new Somvas.GameObject(400, 300);
        rect.addComponent(new Somvas.Rectangle(100, 100, 'blue'));
        scene.add(rect);
        
        // Set and start the scene
        Somvas.setScene('main');
        Somvas.start();
    </script>
</body>
</html>

Asset Loading Example

An example that loads an image and displays it as a sprite.

<!DOCTYPE html>
<html>
<head>
    <title>Somvas Asset Loading</title>
    <script src="https://cdn.jsdelivr.net/gh/cabdiwaaxid-so/somvas@main/index.js"></script>
</head>
<body>
    <canvas id="gameCanvas"></canvas>
    
    <script>
        // Initialize the framework
        Somvas.init('gameCanvas', 800, 600);
        
        // Load an image
        Somvas.AssetLoader.loadImage('player', 'https://placekitten.com/100/100')
            .then(() => {
                // Create a scene
                const scene = Somvas.createScene('main');
                scene.background = '#f0f0f0';
                
                // Create a game object with a sprite component
                const player = new Somvas.GameObject(400, 300);
                player.addComponent(new Somvas.Sprite(Somvas.AssetLoader.get('player'), 100, 100));
                scene.add(player);
                
                // Set and start the scene
                Somvas.setScene('main');
                Somvas.start();
            })
            .catch(error => {
                console.error('Error loading image:', error);
            });
    </script>
</body>
</html>

Troubleshooting & FAQs

Canvas sizing issues

If your canvas isn't displaying correctly, make sure to:

  • Set the canvas dimensions via CSS or JavaScript
  • Call Somvas.init() after the DOM is fully loaded
  • Check that the canvas element exists in the DOM
/* CSS approach */
#gameCanvas {
    width: 800px;
    height: 600px;
    border: 1px solid black;
}

Mobile touch events not working

For touch events to work properly on mobile:

  • Ensure the viewport meta tag is set: <meta name="viewport" content="width=device-width, initial-scale=1.0">
  • Check that your touch handlers are added in the component's update method
  • Test on actual mobile devices as browser emulators may not fully replicate touch behavior

CORS issues with asset loading

When loading assets from different domains, you may encounter CORS issues:

  • Host assets on the same domain as your game
  • Configure your server to send proper CORS headers
  • For development, use a local server instead of file:// protocol

Performance optimization tips

To improve performance:

  • Limit the number of active GameObjects
  • Use object pooling for frequently created/destroyed objects
  • Minimize the use of expensive operations in the update loop
  • Use the appropriate collider type (CircleCollider is less expensive than BoxCollider)

Debug mode not showing colliders

If debug mode isn't visualizing colliders:

  • Make sure you've called setCollider() on your GameObjects
  • Check that Somvas.toggleDebug() is called after the scene is set up
  • Verify that your GameObjects have visible components (debug visuals may be hidden behind them)

Game loop stops unexpectedly

If your game loop stops:

  • Check for unhandled exceptions in your update methods
  • Ensure you're not calling Somvas.stop() somewhere unintentionally
  • Verify that your scene has active GameObjects (empty scenes may be optimized out)
